### 🖨️ પ્રિન્ટ વાઉચર ઓપ્શન્સ (Print Options)
જ્યારે તમે કોઈ એન્ટ્રી સેવ કરશો ત્યારે તમને પ્રિન્ટ કરવા માટે પૂછવામાં આવશે:
- **A4 Single**: આખા A4 પેજમાં બેંક રેકોર્ડ માટેનું મોટું વાઉચર પ્રિન્ટ થશે.
- **A4 3-in-1**: એક જ A4 પેજમાં ઊભી કાતરની નિશાની અને તૂટક રેખા (dotted line) સાથે ત્રણ વાઉચર પ્રિન્ટ થશે:
  1. **Bank Copy (બેંક નકલ)**
  2. **Borrower Copy (ગ્રાહક નકલ)**
  3. **Packet Copy (પોટલી / તિજોરી નકલ)**
